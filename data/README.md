# Datasets

This project evaluates the proposed **SpecTrans-iSTFT-WGAN-GP** architecture on two publicly available datasets:

- **MIT-BIH Arrhythmia Database**
- **ESC-50 Environmental Sound Dataset**

The notebooks automatically download and preprocess the required datasets during execution.

No manual dataset download is required unless you wish to use local copies.

If using local datasets, update the dataset paths in the notebooks accordingly.
